# Nixos Hyprland

Nixos config 4.0; now on a virtual machine (on my main laptop).

## ABOUT

Minimal templates (basically just default `nixos-generate-config`).
I use hyprland as WC, because it has good home-manager integration
(have fully used home-manager as hyprland config, btw).

It works pretty good, and I'm starting to understand most of
the stuff in the files. I want to keep as much as possible within
nix-script (preferably home-manager).
